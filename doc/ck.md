Разрабатывал SPA-панель управления товарами для маркетплейсов (Ozon, Яндекс.Маркет, AliExpress).

Реализовал интерфейсы для массового редактирования цен/остатков/описаний с асинхронной отправкой и отображением статусов выполнения.

Интегрировал frontend с REST API маркетплейсов, работал с динамическими формами и таблицами.
