Внутренний продукт для сотрудников компании (~220 пользователей).

- Разработал универсальный табличный модуль на Vue 3: конфигурация колонок, фильтрации и поведения полностью задаётся с сервера, без хардкода на клиенте.
- Перенёс загрузку данных, фильтрацию, сортировку и пагинацию таблиц на серверную сторону, устранив загрузку полного набора данных в браузер и упростив масштабирование интерфейсов.
- Участвовал в проектировании SPA-архитектуры с использованием Composition API и Pinia, разделяя UI, бизнес-логику и слой данных.
- Мигрировал существующее состояние приложения с Vuex на Pinia, одновременно перевёл код с Options API и JavaScript на Composition API и TypeScript.
- Переписал Telegram-бота с JavaScript + Telegraf на TypeScript + NestJS + Grammy, структурировав код через модули и DI для упрощения поддержки и расширения функциональности.
- Реализовывал новые фичи и исправлял продакшен-баги, включая изменения бизнес-логики и интеграций с backend-API.

Технологии: Vue 3, Composition API, TypeScript, Pinia, REST API, Vite, Node.js, Express, PostgreSQL, Docker, Docker Compose.
